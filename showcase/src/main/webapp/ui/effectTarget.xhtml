Effect Target

By default, effects apply to their parents, this can be configured with the for attribute that represents an effect target.


Source

<h:outputLink id="lnk" value="javascript:void(0)">
	<h:outputText value="Show the Barca Temple" />
	<p:effect type="puff" event="click" for="img">
		<f:param name="mode" value="'show'" />
	</p:effect>
</h:outputLink>
